== Description ==

WP-lightpop plugin is used to overlay images and videos (YouTube, Dailymotion, nicovideo...) on the current page.

== Installation ==

1. Upload the entire `wp-lightpop` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

The control panel of WP-lightpop is in 'Settings > LightPop'.
