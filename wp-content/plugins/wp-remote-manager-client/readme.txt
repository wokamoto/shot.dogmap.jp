This plugin enables the web application "WP Remote Manager" to manage your sites.

== Description ==
This plugin enables the web application [WP Remote Manager](http://wp.remotemanager.me/) to manage your sites using OAuth authorization for connection between the web application and your sites.

= Features =

* Unifying management of multiple WordPress sites.
* You can see the status of tens or hundreds of your WordPress sites at once.
* You can update multiple WordPress sites at once very easily.
* Full backup with one click.
* Manage your sites securely and easily.

== Installation ==

1. Install The "WP Remote Manager Client" either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin.
3. Sign up for an account at [WP Remote Manager](http://wp.remotemanager.me/) and add your site.
